Chapter 9: Introduction to High Level Languages

List four disadvantages of using assembly language:

  1. Manage the movement of data items between and among memory locations and registers.
  2. The programmer must take a microscopic view of a task, breaking it down into tiny subtasks that direct what is going on in individual memory locations and registers.
  3. An assembly language program is machine specific.
  4. Satements are not natural-language-like.

List four expectations programmers have for writing programs in a high-level language ("third generation" language):

  1. The programmer need not manage the details of the movement of data items within memory or pay any attention to exaclty where those items are stored.
  2. The programmer can take a microscpoic view of tasks, thinking at a higher level of problem solving.
  3. Programs are portable rather than machine specific.
  4. Programming statements are closer to natural language and use standard mathematical notation.

Briefly describe or define:

  1. Object code: System software that translates assembly languge instructions into binary machinelanguage instructions.
  2. Source code: High-level language instructions
  3. Procedural language: Consists of sequences of statements that manipulate data items.
  4. Integrated development environment (IDE):Lets programmers perform a number of tasks within the shell of a single application program, rather than having to use a separate program for each task.
  5. Software development life cycle: The overall sequence of steps needed to complete a large-scale software project.
  6. Waterfall model of software development: Software development is a completely sequential process.
  7. Agile model of softwawre development: A whole suite of methods and processes that helo; to promote a flexible and ready response to meet a shifting and ever-changing target.

Describe each of these components of the software development life cycle:

  1. Feasibility study: Evaluates a proposed projject and compares the costs and benefits of various solutions.
  2. Problem specification: Involves developing a clear, concise, and unambiguous statement of the exact problem the software is to solve.
  3. Program design: When it's clear what is to be done, the program design phase is the time to plan how it is to be done.
  4. Algorithm selection: Once the various subtasks have been identified, algorithms must be found to carry them out.
  5. Coding: is the process of converting the detailed designs into computer code.
  6. Debugging: is the process of locating and correcting program errors, and it can be a slow and expensive operation that requires as much effort as writing the program in the first place.
  7. Testing: Test data that is carefully chosen to exercise all the different logic paths through a program can help uncover errors.
  8. Verification: can be used to prove that if the input data to a program satisfies certain conditions, then, after the program has been run on this data, the output data satisfies certain other conditions.
  9. Documentation: all of the written and online material that makes a program understandable.

Questions & Answers:

Q: Why does the textbook refer to "the Tower of Babel" regarding computer languages?

A:Because there are multiple programming languages and not one that's universal to everyone.


Q: Why isn't there just one computer language?

A:Each language was designed to meet the specific needs of one particular are of application.


Q: Why didn't FORTRAN use the less than symbol when comparing two numbers?

A:The keypunches that were used to create the punced cards on which early Fortran programs were submitted to the computer had no such symbol.


Q:What was COBOL designed for?

A:COBOL was designed to serve business needs such as managing inventories and payrolls.


Q:What was the language, C, designed for?

A:It was originally designed for systems programming, in particular for writing the operating system UNIX.


Q:What is the difference between C and C++?

A:C++ provides the ability to do object oriented programming.


Q:Who or what is the language, Ada, named for and why?

A:After Ada Augusta Byton Lovelace. Ada Lovelace is regarded as the world's first programmer on the basis of her correspondence with Charles Babbage and her published notes on his work with the Analytical Engine.


Q:What is a Java "applet"?

A:Small applications designed to run from webpages.


Q:How much does Python cost to download, install, and use?

Q:What is Microsoft.NET?

A:It's essentially a giant collection of tools for software development.


Q:What is SQL and what is it used for?

A:SQL is designed to be used with databases, which are collections of related facts and information.


Q:What is R and what is it used for?

A:R is a specialized programming language designed for statistical computing and graphics.


Q:What is Go (or golang) and what is it used for?

A:Go is an open source language; the source code is available for anyone to examine and use, and changes or additions can be proposed. Two of the target areas for Go applications are systems programming and programming for multicore machines.


Q:What is Swift and what is it used for?
